## ng-threeparty

Build with
- Angular 19
- Nx 20
- Tailwindcss
- Material UI

add material:
> pnpm npx nx g @angular/material:ng-add --project=ng-threeparty

add tailwindcss:
> pnpm npx nx g @nx/angular:setup-tailwind ng-threeparty
